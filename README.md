# SmartBudget

Простой веб-сайт для учёта доходов и расходов.

## Что сделано

- Страница `index.html` с формой для добавления транзакций
- Сохранение данных в `localStorage`
- Итоги по доходам, расходам и балансу
- Удаление отдельных записей и очистка всех данных
- Стилизация в `styles.css`
- Логика приложения в `script.js`
- Персональная страница `profile.html`
- Переключение языка интерфейса (RU/EN)

## Запуск

Просто откройте `index.html` в браузере.
